「TODOの残件」を聞かれたときは以下の手順で対応する:

1. `TODO.md` と `TODO_NEXT.md` の両方を確認する（grep で ⚠️/❌ を抽出）
2. **優先度が高く難易度が低い順**でリストアップして提案する
3. 大規模・複雑なものは後回し、small/medium で即着手できるものを先に出す

**Why:** 毎回どちらか片方しか見ていなかった。また難易度を考慮せず羅列していた。
